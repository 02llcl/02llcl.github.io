暑假集训杭电第九场

发表于 2021-08-18 分类于 2021暑假集训 阅读次数：

前言

这次比赛感觉来了,,算最好的几次了

真不容易

 

1002 Just another board game

题目链接

大意:博弈题,两个人轮流在棋盘上移动,在每个回合中，如果轮到Roundgod移动，他可以设置c为某个满足1≤c≤m的任意值，如果是木神移动，他可以设r为某个满足1≤r≤n的任意值.

只需要考虑最后两步,因为无论怎么走最后都能改变.

并且需要根据k的奇偶性来决定最后找最大值还是最小值.

还需要跟开头位置进行比较,因为存在原地结束的情况

 

1003 Dota2 Pro Circuit

题目链接

给每个队伍分配名次,计算每个队伍最后总分的排名

利用贪心即可

 

1007 Boring data structure problem

题目链接

大意:模拟一个数据结构,进行访问操作添加,可以查询中位数

解:使用双项链表维护从左和从右,显然每次L,R操作的复杂度为O(1)

删除只需标记删除即可

加一个中间值的链表指针即可进行查询

大模拟

 

1010 Unfair contest

题目链接

大意:给定分数使上面的an减bn最小,同时满足a的总分超过b的总分

排序,可以知道给定a的有效分数和b的有效增加分数范围

接着利用总分的差值进行分类即可

还需要特判n==1等情况
